『ムーセとジン』
一次創作のキャラクターをつついたり構ったりするためだけのゴーストです。
いじり続けても好感度が上がるわけではないです。

顔をダブルクリックでメニューが開きます。
それ以外のところをつつくとなんかしゃべります。
頭をカーソルでなでなでするとなんかしゃべります。


作った人：ナウシロ
